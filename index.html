    My Projects

    Notes Maker Webapp

    Technologies Used: Python, Django, HTML, Bootstrap CSS, Azure App Service, Github Actions

    • Created a Web application using Django to take notes on the go with simple and user friendly UI.
    • Used the Django-Admin for User Authentication and User Management for saving notes of each user separately.
    • Implemented CRUD operations to save, delete, edit and retrieve the notes to/from SQLite Database.
    • Used Github Actions CI/CD platform to deploy the app to Azure App Service on Linux.




    Facenforce

    Technologies Used: Node.js, Faceapi.js, HTML, CSS, JavaScript, Github

    • Developed a face-recognition web-app using faceapi.js, a pre-trained model and node.js to detect and recognize multiple faces in an image.
    • Implemented labled-image upload with folder structure for face registration to simplify the client side UI.
